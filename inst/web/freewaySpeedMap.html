freewaySpeedMap

Usage

freewaySpeedMap(con, corridorID, startDate, endDate, weekdays = TRUE, outputPng = NULL)

Arguments

con
database connection PORTAL PostgreSQL database
corridorID
ID number of the corridor to plot. See 'corrdidors' table to select ID for plotting.
startDate
Start date of data to query (YYYY-MM-DD format)
endDate
End date of data to query (YYYY-MM-DD format)
weekdays
Boolean indicating whether to subset data to weekdays. Defaults to TRUE.
outputPng
.png file path of output plot. Default is NULL; if NULL plots to current device (i.e. RStudio plot device).

Value

None

Description

Produces a speed-based heatmap (AKA brainscan plot) for a corridor with PORTAL's freeway data system. See Freeway Speed Map vignette for example usage.
