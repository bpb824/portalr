Query and return freeway loop and radar data from PORTAL

Usage

freewayData(con, dets, startTime, endTime)

Arguments

con
Database connection object.
dets
Vector of detector IDs.
startTime
Start time of requested data in either a YYYY-MM-DD or YYYY-MM-DD HH:MM:SS format.
endTime
End time of requested data in either a YYYY-MM-DD or YYYY-MM-DD HH:MM:SS format.

Value

Result of freeway data query.

Description

Query and return freeway loop and radar data from PORTAL
