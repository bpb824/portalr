Aggregate PORTAL data over time.

Usage

aggTime(unAgg, aggVars, timeCut, hod = FALSE)

Arguments

unAgg
Data that you want to aggregate.
aggVars
Variables you want to retain in aggregation, such as 'lanenumber' or 'stationid'
timeCut
A string indicating how you want to aggregate over time. Examples include '1 min', '5 min', '1 hour', '1 day', etc.
hod
A boolean indicating weather you want to aggregate by hour of day across multiple days.

Value

Aggregated data.

Description

Aggregate PORTAL data over time.
