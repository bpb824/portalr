Filter freeway data using threshold values

Usage

filterFreeway(fdata)

Arguments

fdata
Your raw freeway data.

Value

Filtered freeway data

Description

This function will use six criteria developed for filtering PORTAL data to filter your raw data. The criteria are summmarized on Table 1 of this document: http://portal.its.pdx.edu/Portal/static/files/fhwa/Freeway
